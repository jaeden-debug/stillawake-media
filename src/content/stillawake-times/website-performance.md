## Core Web Vitals

Google introduced Core Web Vitals to measure real-world user experience quality.

The three primary metrics are:

- Largest Contentful Paint
- Interaction to Next Paint
- Cumulative Layout Shift

These measure:

- loading speed
- responsiveness
- visual stability

Core Web Vitals now directly affect SEO.

## Largest Contentful Paint (LCP)

LCP measures how quickly the main visible content loads.

Poor LCP commonly comes from:

- oversized images
- render-blocking resources
- weak servers
- slow JavaScript
- poor caching

Fast LCP improves:

- engagement
- perceived speed
- SEO
- user confidence

## Interaction to Next Paint (INP)

INP measures responsiveness.

This evaluates how quickly a page reacts to user interactions.

Poor responsiveness creates experiences that feel sluggish.

Heavy JavaScript often causes INP issues.

## Cumulative Layout Shift (CLS)

CLS measures layout stability.

Users hate unstable interfaces.

Unexpected shifting creates frustration and distrust.

Good performance systems maintain visual stability continuously.
